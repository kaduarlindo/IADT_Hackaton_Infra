🔥 Começando Agora

Opção 1: Desenvolvimento Local (Recomendado)

chmod +x start-dev.sh ./start-dev.sh # Acesse: http://localhost:3001

Opção 2: Deploy no AWS

cd terraform terraform init terraform plan terraform apply # Siga as instruções em docs/DEPLOYMENT.md

Opção 3: Testar API

chmod +x test.sh ./test.sh https://seu-api-endpoint/dev

📁 Estrutura do Projeto

Arquitetura Hackaton/ ├── terraform/ # Infraestrutura AWS │ ├── main.tf # Configuração principal │ └── modules/ # S3, DynamoDB, Lambda, IAM, CloudFront ├── src/ # Backend (Python) │ ├── domain/ # Entidades de domínio │ ├── ports/ # Interfaces abstratas │ ├── application/ # Casos de uso │ └── adapters/ # Implementações ├── frontend/ # Frontend (React) ├── docs/ # Documentação ├── docker-compose.yml # Dev environment └── Makefile # Comandos úteis

🎯 Recursos Principais

  • ✅ API Gateway + Lambda serverless
  • ✅ DynamoDB para dados estruturados
  • ✅ S3 para armazenamento de arquivos
  • ✅ Geração de PDFs com ReportLab
  • ✅ Frontend React moderno
  • ✅ CloudFront para CDN
  • ✅ IAM com permissões mínimas
  • ✅ CloudWatch + X-Ray monitoring
  • ✅ CORS habilitado
  • ✅ Docker para desenvolvimento local

🛠️ Ferramentas & Tecnologias

Backend

Python 3.11 com arquitetura hexagonal

Frontend

React 18 + Tailwind CSS

Infraestrutura

Terraform + AWS

Desenvolvimento

Docker + docker-compose
